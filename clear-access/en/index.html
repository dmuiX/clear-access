Technical background

Tools
See the technical background

I place a central access layer in front of your tools. It decides who can reach which tool. Your applications remain unchanged — only the access point becomes consistent.

  • Identity: Authelia with OIDC/OAuth2 for SSO
  • Directory: Samba-DC / LDAP / AD, MS Entra ID, Google Workspace
  • Gateway: SWAG / Nginx reverse proxy + SSL
  • MFA: 2FA across critical tools
  • Access rules: clear role mapping and offboarding

No migration of users or data. The goal is a clear access state — not a rebuild.
